Bienvenido al juego Encriptador/Desencriptador de texto.
*****************************************************************************************************************
Este juego trata de escribir palabras, frases o parrafos es libre para que el jugador decida cómo jugarlo. 
Para que todo funcione bien el jugador deberá omitir palabras en mayúsculas y no usar los siguientes signos
de puntuación ni caracteres especiales: $.¿?~!\¡@#%^&*()_|}{[]><:" `;,\u0300-\u036f']/g

Aventúrate a mi juego y diviértete con tus amigos retándolos a descifras tu mensaje secreto. 
Para facilitarles que descifren tu mensaje, te invitamo a usar las siguientes llaves de encriptación:
****************************************************************************************************************
Las "llaves" de encriptación que utilizaremos son las siguientes:

La letra "e" es convertida para la tecla "enter"
La letra "i" es convertida para la palabra "imes"
La letra "a" es convertida para la palabra "ai" 
La letra "o" es convertida para la palabra " ober" 
La letra "u" es convertida para la palabra "ufat"

*****************************************************************************************************************
Espero que lo disfrutes y seas libre de tomar mi código y hacerles todos los cambios que desees, 
sólo recuerda compartirme tu versión final para que también pueda jugar y divertirme.

Saludos.
